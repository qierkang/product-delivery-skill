# AI+RPA Demo 阶段最小验收清单

## 必须完成（Demo 通过门槛）

- [ ] Mock 系统可在浏览器打开，有完整的登录页、订单列表页、派单弹窗
- [ ] 决策引擎服务在本地启动，`POST /recommend` 接口返回正确推荐结果
- [ ] 影刀脚本可完整走通：登录 → 抓单 → 调引擎 → 派单 → 验证状态
- [ ] 至少 10 笔端到端自动派单，成功率 ≥ 80%
- [ ] 有派单日志文件（CSV），包含时间、订单号、推荐师傅、得分、结果

## 加分项（不强制）

- [ ] 有 Demo 录屏或关键步骤截图
- [ ] 决策引擎响应日志（含每师傅细分得分）
- [ ] 告警触发演示（模拟登录失败场景）

## 不在 Demo 范围内

- Session 保活机制（Production 阶段）
- 幂等防重派单（Production 阶段）
- 告警接入钉钉/邮件（Production 阶段）
- 7×24 稳定性测试（Production 阶段）
