## 启动顺序说明（供验收参考）

1. `start-demo.sh` → Docker 启动 PostgreSQL
2. Docker init 运行 `drawing-init.sql` → 建业务表 + 种 20 条 drawing + 20 条 price + FK 回填
3. Spring Boot 启动 → Flyway 运行 V1__（系统表）→ V1_1__（种用户/角色/菜单基础数据）→ V1_2__ → V1_3__（图纸模块菜单）
4. 登录后侧边栏出现「采购估价」目录，四个子页面可访问
