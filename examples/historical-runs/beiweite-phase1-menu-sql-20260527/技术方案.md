## drawing-init.sql 修改位置

文件末尾追加：
```sql
-- 回填 price_record.drawing_id FK（避免悬空）
UPDATE price_record pr
SET drawing_id = di.id
FROM drawing_info di
WHERE di.drawing_no = pr.drawing_no
  AND pr.drawing_id IS NULL;
```

## system_menu INSERT 字段顺序

参照 V1_1__init_seed_data.sql 中 system_menu 的已有格式：
```
("id","name","permission","type","sort","parent_id","path","icon","component","component_name",
 "status","visible","keep_alive","always_show","creator","create_time","updater","update_time","deleted","application_scope")
```

- status = 0（正常）
- visible = 't', keep_alive = 't', always_show = 't'
- creator/updater = 'admin'
- deleted = 0
- application_scope = 'ADMIN_WEB'
- 父目录 component 和 component_name 填 NULL
