## 1. 信息架构

## 2. 风格方向与设计策略

- 风格方向：`结果揭晓页 + 轻品牌活动页 + 运营看板`
- 页面气质关键词：`红镜、神秘、开奖感、轻戏剧性`
- 品牌 / 调性说明：前台强调情绪价值和分享欲，后台强调运营可读性
- 主操作与危险操作层级：首页进入测试、结果页分享 / 留资 / 重测为主操作；后台危险动作弱化
- 信息层级原则：前台先结果后说明，后台先指标后明细

## 3. 设计 Token 与组件策略

- 主色 / 辅色方向：深红、暖金、黑金背景、柔白卡片
- 字体 / 字重方向：标题杂志感，正文清晰，结果标题更强
- 圆角 / 阴影方向：前台卡片圆角更明显，后台卡片克制
- 列表 / 卡片 / 表单组件策略：前台结果区以大卡片 + 标签为主，后台以数据卡片 + 表格为主

## 5. 关键交互

| 场景 | 触发动作 | 系统反馈 | 成功状态 | 失败状态 |
|---|---|---|---|---|
| 首次进入 | 先展示邮箱弹层，可跳过 | 弹层平滑进入，不阻塞关闭 | 进入昵称层或首页主 CTA | 关闭后仍可继续测试 |
| 生成昵称 | 默认给一组随机昵称 | 点击“换一个”即时刷新 | 昵称刷新成功 | 保留当前昵称 |
| 自定义昵称 | 用户输入后覆盖随机值 | 即时校验长度与字符 | 昵称通过校验 | 就近提示错误 |
| 开始测试 | 点击进入 `/test` 并保留 session | 按钮 loading + 禁重 | 成功进入测试页 | 留在原页并提示 |
| 选择答案 | 单题单选，已选高亮 | 下一题按钮可用 | 进入下一题 | 保留当前题 |
| 提交测试 | 未答完不可提交 | 显示缺失题目提示 | 进入报告生成过程 | 提示未完成 |
| 报告展示 | 报告页进入时播放轻量烟花 / 礼花特效 | 不遮挡主要信息与按钮 | 报告首屏加载完成 | 兜底空态 |
| 海报生成 | 结果页加载后异步生成 | 先展示骨架，后展示图片 | 海报可预览 | 可重试 |
| 留邮箱发报告 | 结果页也可再次填邮箱 | 成功后 toast + 状态更新 | 留资成功 | 表单下方显示失败原因 |
| 解锁详细报告 | 点击后进入 detail 页 | 未开支付则展示“即将开放” | 进入详细报告页 | 保持当前页并提示 |
| 重新测试 | 清空当前答题态，保留历史报告 | 二次确认后执行 | 返回测试起点 | 取消后保留当前结果 |

## 6. 状态与异常

| 状态类型 | 展示规则 | 用户可执行动作 |
|---|---|---|
| loading | 骨架屏、按钮 loading、进度动画 | 等待或返回 |
| 报告生成中 | “人格镜像生成中”过渡页 | 等待 |
| 海报生成中 | 海报占位卡 + 重试按钮 | 等待 / 重试 |
| 空结果 | 兜底提示 + 返回重测按钮 | 返回重测 |
| 无权限 | 后台登录失效时提示重新登录 | 重新登录 |
| 接口失败 | toast 错误 + 重试按钮 | 重试 / 返回 |

## 7. 动效与反馈

- 页面切换：首页到测试页、测试页到报告页使用轻过渡，不拖慢节奏
- 弹层 / 抽屉：邮箱弹层、昵称弹层进入平滑但不阻塞关闭
- 提交 / 保存 / 删除：按钮 loading + toast / inline message
- 长任务 / 异步生成：报告生成、海报生成必须展示阶段性提示

## 8. 多端适配

| 端 | 布局策略 | 组件差异 | 手势/交互差异 |
|---|---|---|---|
| Web | 居中容器，兼容宽屏展示 | 结果卡和后台卡片可更大 | 鼠标 hover 只做辅助，不是唯一反馈 |
| APP | 预留 | 结果页可强化沉浸感 | 预留 |
| 小程序 | 预留 | 更轻量的结果页与留资交互 | 预留 |
| H5 | 单列滚动，底部 CTA 可触达 | 弹层、卡片、按钮优先单手可用 | 滑动和点击优先 |

## 9. 可访问性与响应式要求

- 断点策略：H5 主适配 `375-430`，Web 兼容 `1100+`，后台优先 `1280+`
- 触控热区要求：前台 CTA、选项卡、重测按钮不小于 `44x44`
- 键盘 / 焦点 / 错误提示要求：后台表单和登录框需要清晰焦点态；邮箱校验错误就近展示
- 长文本 / 文案溢出处理：人格标签、昵称、邮件错误文案需兼容换行

## 10. UI验收标准

- 视觉还原度：前台符合“红镜实验室 + 开奖感”，后台符合“运营看板”风格
- 交互一致性：主流程 `开始 -> 作答 -> 提交 -> 结果 -> 留资 -> 重测` 可完整走通
- 文案与提示一致性：错误提示、锁定态提示、邮箱提示口径统一
